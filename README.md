# storybook-test

https://masato-ohira.github.io/storybook-test/

storybookの実験用リポジトリ

## keywords

storybook, webpack, vue

## 動作環境


Node.jsのインストールが必要です。

https://nodejs.org/ja/

* 開発時バージョン 8.11.2

## 依存モジュールのインストール

```
npm install
```

## 使い方

storybook起動

```
npm run storybook
```

storybook静的ビルド

```
npm run buildbook
```
